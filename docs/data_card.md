\# Data Card: Spatiotemporal Wildlife Dataset



\## Dataset Overview



This project uses the Spatiotemporal Wildlife Dataset from Kaggle for camera-trap imagery and wildlife detection experimentation.



\## Dataset Source



https://www.kaggle.com/datasets/travisdaws/spatiotemporal-wildlife-dataset



\## Data Type



The dataset contains camera-trap imagery with species and location-related metadata where available.



\## Use Case



The dataset supports wildlife detection, geospatial analytics, and cloud-native ML pipeline evaluation.



\## Data Considerations



Potential considerations include:



\- Class imbalance across species

\- Variable image quality

\- Different lighting and environmental conditions

\- Geographic sampling bias

\- Metadata completeness

\- Species-level label consistency



\## Responsible Use



Location-aware wildlife data should be handled carefully to avoid exposing sensitive ecological information, protected species locations, or vulnerable habitat areas.
